# IntelliCure-AI_SmartGlove_Health_Ecosystem
This is an Open Source project based on Arduino , AI , Machine learning , IOT based interface which comprises of multiple sensors to make a comfortable and day-2-day use gadget for critical patients , to monitor health regularly and to provide emergency services faster without fail.

This project is a Multidisciplinary project that comprises of teammates from different branches of engineering to come together and work on an innovative solution as a first year engineering students. and to learn how to make products work and how things interact with the real world and to apply our knowledge in real life usecases.
